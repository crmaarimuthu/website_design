```bash
# 1. Install dependencies
npm install

# 2. Test local build
npm run build
# Expected: dist/ folder created without errors

# 3. Preview production build
npm run preview
# Visit http://localhost:4173
# Expected: App loads perfectly

# 4. Lint check
npm run lint
# Expected: No critical errors

# 5. Git verification
git status
# Expected: No uncommitted changes
```

## 🔐 Environment Variables Setup

### Local Development
1. Create `.env.local` (copy from `.env.example`)
2. Add your variables:
   ```env
   VITE_API_URL=http://localhost:3000
   VITE_APP_NAME=My App
   ```
3. Restart: `npm run dev`

### Production (Vercel)
1. Vercel Dashboard → Settings → Environment Variables
2. Add variables (must start with `VITE_`)
3. Redeploy or push new commit

**Important:** Variable names **MUST** start with `VITE_` to work in Vite
